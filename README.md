# MDMS Super Admin

Minimal MDMS admin panel.

## Features

- Login with existing MDMS account
- Dashboard cards
- Searchable table views
- CSV export
- Mobile responsive layout

## Tables

- Clinics
- Staff
- Patients
- Appointments
- Visits
- Invoices
- Payments
- Files
- Invites
- Website leads

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Cloudflare Pages

